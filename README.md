# Secure-Sensitive-Data-Processing-Platform

The project focuses on building a secure data processing platform for climate data, integrating key security measures. The system is architected using Flask-based APIs for parameterized data requests, containerized with Docker for efficient deployment. Databricks handles scalable data processing and storage in a Delta format.

Key Security Features:

Data Encryption: 
Sensitive data (e.g., email addresses, phone numbers) is encrypted using advanced cryptography modules to ensure confidentiality and integrity.

Unity Catalog: 
Employed for data lineage and governance, enabling secure data sharing within and outside the organization. Future enhancements include PII flagging for automated encryption compliance.
This platform ensures robust data governance, secure access, and confidentiality, critical for climate-related data.
